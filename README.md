# 📊 Linear Regression Project

## 📌 Overview
This repository demonstrates a beginner-friendly implementation of **Linear Regression** using Python. The project predicts salaries based on years of experience, showcasing how machine learning models can be applied to real-world datasets.

## 📂 Repository Structure
- **`linearregression.ipynb`** → Jupyter Notebook containing the step-by-step implementation of the model.  
- **`Salary_Data.csv`** → Dataset with years of experience and corresponding salaries.  
- **`README.md`** → Documentation explaining the project, workflow, and usage.  
- **`.gitignore`** → Configured for Python projects to exclude unnecessary files.  

## ⚙️ Technologies Used
- Python 3.x  
- Jupyter Notebook  
- Libraries:  
  - `numpy`  
  - `pandas`  
  - `matplotlib`  
  - `scikit-learn`  

## 🚀 Workflow
1. **Data Loading** → Import the dataset (`Salary_Data.csv`).  
2. **Exploratory Data Analysis (EDA)** → Visualize relationships between experience and salary.  
3. **Model Training** → Apply Linear Regression using `scikit-learn`.  
4. **Model Evaluation** → Assess performance with metrics like Mean Squared Error (MSE) and R² score.  
5. **Prediction** → Predict salaries for new data points.  

## 📈 Results
- The model successfully captures the linear relationship between years of experience and salary.  
- Visualizations show the regression line fitting the dataset.  

## 🧑‍💻 How to Run
1. Clone the repository:  
   ```bash
   git clone https://github.com/dishaK26/ML-Projects.git
   ```
2. Navigate to the project folder:  
   ```bash
   cd ML-Projects
   ```

   ```bash
   jupyter notebook linearregression.ipynb
   ```
